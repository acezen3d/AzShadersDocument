# Az Matcap shaders
v1.0.0

## Introduction
Az Matcap shaders are a better alternative to `Goo`, `BetterMatcap`, and other matcap-based shaders in Koikatsu. These shaders integrate seamlessly with other Az shaders, providing a cohesive experience. Based on an in-depth study of the matcap feature, these shaders offer a refined and optimized solution for matcap rendering.
