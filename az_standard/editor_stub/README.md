# Editor stub shaders

Shaders in this directory are stubs of Az Standard shaders that can be used in Unity Editor for modding. You can download them and place them in your Unity project so that the shaders will appear in Unity's shader selection dropdown. Please refer to the [tutorial](../../tutorials/auto_replace_shaders_from_asset_bundles_in_unity_editor_play_mode.md) to learn how to use them.\
Happy modding!
